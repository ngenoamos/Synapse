# Cathedral Scanner Frontend

Live blockchain wallet scanner with Tailwind CSS.

**Live Demo:** https://ngenoamos.github.io/Synapse/
**API Backend:** https://cathedral-scanner-api.onrender.com

## Features
- Real-time wallet scanning (Ethereum & BSC)
- SRS risk scoring (0-100)
- Responsive design with Tailwind CSS
- Direct API integration

## Development
Built with HTML5, Tailwind CSS, Vanilla JS
